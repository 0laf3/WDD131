Movie Vision Site Plan - WDD131

Movie Vision

The name was chosen to reflect the website's focus on providing a comprehensive view of the movie world. It suggests a platform for discovering and exploring movies, getting recommendations, and staying informed about the latest in the film industry.

Site Purpose

Clearly defines the aim of the website, such as offering movie reviews, trailers, news, and personalized recommendations.

Scenarios

  • Display information on popular movies
  • Display information about celebrities
  • Crowdsource ratings
  • Review not from one person but from a bunch critics.
  • Speculate on the type of audience who may like this film.
